## 代码解析

### 元组的特点

1. **不可变性**: 元组创建后不能修改
2. **有序性**: 元素有固定的顺序
3. **可包含不同类型**: 可以存储不同数据类型的元素

### 关键代码说明

```python
# 创建元组
my_tuple = (1, 2, 3, 4, 5)

# 访问元素(通过索引)
first_element = my_tuple[0]

# 切片
sub_tuple = my_tuple[1:4]
```

## 常见问题

**Q: 为什么使用元组而不是列表?**  
A: 元组不可变,更安全且性能更好,适合存储不需要修改的数据。

**Q: 如何创建只有一个元素的元组?**  
A: 需要在元素后加逗号,如 `(1,)`
