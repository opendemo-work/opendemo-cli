# Python数组操作实战演示

## 简介
本项目通过两个具体场景，帮助初学者理解Python中数组（即列表）的常见操作。虽然Python没有传统意义上的“数组”类型，但其内置的列表（list）提供了类似且更灵活的功能。本演示涵盖基础增删改查、遍历以及数值处理等核心技能。

## 学习目标
- 理解Python中列表作为数组的用法
- 掌握列表的创建、访问和修改方法
- 学会使用循环处理数组元素
- 了解如何对数组进行统计计算（如求和、平均值）
- 培养编写可读性强、结构清晰代码的习惯

## 环境要求
- Python 3.6 或更高版本
- 操作系统：Windows、macOS、Linux 均支持

## 常见问题解答

**Q1: Python中有真正的数组吗？**
A: 标准Python中没有固定类型的数组，但列表（list）可以动态存储任意类型数据，行为类似于数组。若需高性能数值计算，推荐使用NumPy库中的数组。

**Q2: 为什么不用 array 模块？**
A: `array` 模块提供的是底层数值数组，功能有限。对于大多数初学者场景，列表更易用、更直观。

**Q3: 运行时报错 'python not found' 怎么办？**
A: 请确认已正确安装Python，并将其添加到系统环境变量PATH中。Windows用户建议在安装时勾选“Add to PATH”。
