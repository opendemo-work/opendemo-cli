# Go结构体实战演示

## 简介
本演示项目展示了Go语言中`struct`（结构体）的核心用法，包括结构体定义、方法绑定、字段封装以及结构体组合（替代继承的方式）。适合初学者理解Go面向对象编程的基本模式。

## 学习目标
- 掌握Go中结构体的定义与初始化
- 理解如何为结构体绑定方法
- 学会使用结构体组合实现代码复用
- 理解值接收者与指针接收者的区别

## 环境要求
- 操作系统：Windows、macOS 或 Linux（任意主流发行版）
- Go版本：1.19 或更高（推荐使用最新稳定版）

## 代码解析

### person.go
- 定义了`Person`结构体，包含Name和Age字段
- `Greet()` 方法使用值接收者，适用于只读操作
- `Work()` 方法使用指针接收者，允许修改字段（虽然本例未改，但体现规范）

### employee.go
- `Employee`结构体通过匿名嵌入`Person`实现组合
- 自动获得`Person`的所有字段和方法，实现类似“继承”的效果
- `Display()` 方法展示组合后的信息

### main.go
- 演示结构体的初始化语法：字面量和new
- 调用方法并观察组合行为

## 常见问题解答

**Q: 什么时候使用值接收者，什么时候使用指针接收者？**
A: 如果方法需要修改结构体字段，或结构体较大（避免复制开销），应使用指针接收者；否则可使用值接收者。

**Q: Go有类吗？**
A: Go没有类，但结构体+方法的组合提供了类似面向对象的能力。

**Q: 如何实现继承？**
A: Go不支持继承，而是使用结构体组合（嵌入）来复用代码。
